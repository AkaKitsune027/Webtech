Code From Outside

1. Transition : เมื่อเราใส่ transition จะกำหนดให้ตัวที่เราใส่มีการเปลี่ยนแปลงจาก
value เก่าไป value ใหม่ ของแต่ละ Property ที่กำหนดไว้
หรือพูดง่าย ๆ คือเป็นตัวที่กำหนดให้ element ใน html ค่อย ๆ เปลี่ยน value ในแต่ละ property ทีละนิด ๆ โดยที่ตัว transitions จะสามารถเปลี่ยนสีหนึ่งเป็นสีหนึ่งได้ เปลี่ยนขนาดจากขนาดหนึ่งเป็นอีกขนาด เปลี่ยนตำแหน่งจากตำแหน่งหนึ่งเป็นอีกตำแหน่งหนึ่ง

2. Transform : scale(value) เป็นการปรับขนาดวัตถุ ใส่เป็นค่า (x, y) หรือ ถ้าอยากให้ลดทั้งค่า x, y ในคราวเดียว ให้ระบุค่าเดียวไปเลย โดยไม่ใส่ลูกน้ำ(,)

3. Transform : translate(X, Y) เป็นการย้าย element ไปยัง position ที่กำหนด

4. Animation: mymove (value)s infinite alternate เป็นการกำหนดให้ object(หรือ mymove) ขยับแบบ infinite หรือวนลูป ทุก ๆ value ที่เรากำหนด

5. @Keyframe mymove {from {namePosition: Position that youwant to start animation}
to {namePosition: Position that youwant to end animation}

6. z-index : เป็นตัวที่จัดการการซ้อนทับของ layout ในแกน z

7. Calc() : เป็น 1 ใน CSS Function ที่ใช้ในการคำนวณขนาด CSS Property โดยเฉพาะ ใช้ได้ทั้ง ค่าความยาว ค่าความถี่ มุม จำนวนครั้ง เปอร์เซ็นต์ และค่าตัวเลข เป็นต้น
